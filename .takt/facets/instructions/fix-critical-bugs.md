Piece Context に示された Report Directory 内のレポートを確認し、検出された問題を修正してください。

## 手順

1. 全スキャンレポートを読み、修正すべき問題を把握する
   - ロジックバグレポート: {report:01-logic-scan.md}
   - セキュリティレポート: {report:02-security-scan.md}
   - 並行処理バグレポート: {report:03-concurrency-scan.md}
2. 重大度の高い問題から順に修正する（Critical → High）
3. 各修正後にテストを実行し、リグレッションがないことを確認する
4. 修正不能な問題がある場合は理由を報告する
