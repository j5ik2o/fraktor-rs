## ユーザー入力

```
$ARGUMENTS
```

## 目的
- 追加の方針や制約を `.kiro/steering/` に反映する。

## 手順
1. 追加したい方針・対象範囲・理由を整理する。
2. `.kiro/steering/<name>.md` を作成または更新する。
3. 既存の steering と矛盾しないか確認する。
4. 内容は日本語で簡潔に記述する。
